How to use Git

GIT is a piece of software which tracks any changes when developing software solutions. Along with GitHub which is a place where Repository live (Repos for short), Git as well as GitHub is a must skill for most software develpers

Adding files to staging area

When working on a new file, we may add new text, remove existing comments or even shanging. To track these changes we use the command git add [name_of_file with its extension].
Adding a file to staging area will let the Git track any chnages happen to that file

Commiting changes

[git commit] is another command which is used to add a file or many files to the Repo are and make them reading for pushing to the GitHub
git commit is recommended to be used along with the -m flag which is used to write a short descriptive message about what happened. It is hightly recommended to use the impertaive for for writing messges.
